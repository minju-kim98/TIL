## 14-1 변수 알아보기
### 변수란
### 변수 선언의 규칙
1. 변수 이름은 영어 문자와 언더스코어(_), 숫자를 사용한다.
2. 자바스크립트는 영어 대소 문자를 구별하며 예약어는 변수 이름으로 쓸 수 없다.
3. 여러 단어를 연결한 변수 이름은 중간에 대문자를 섞어 쓴다.
4. 변수 이름은 의미 있게 작성해야 한다.
